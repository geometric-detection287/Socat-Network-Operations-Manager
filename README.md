## 🔍 Troubleshooting

### The app will not open
- Right-click the file and choose **Run as administrator**
- Make sure the download finished
- If the file is in a ZIP folder, extract it first

### The port is already in use
- Pick another port
- Close the app or service that uses the same port

### Nothing connects
- Check the host name or IP address
- Check the port number
- Make sure TCP and UDP match on both sides
- Review your firewall settings

### The task stops after a drop
- Turn on watchdog restart
- Check that the target system is still online
- Verify network access on both ends
